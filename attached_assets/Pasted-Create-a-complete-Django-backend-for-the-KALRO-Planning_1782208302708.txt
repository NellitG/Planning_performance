Create a complete Django backend for the KALRO Planning, Performance Management (PPM) System Projects Module and ensure it integrates with every frontend feature that has been implemented.

### Backend Technology Stack

Use:

* Django
* Django REST Framework (DRF)
* SQLite (`db.sqlite3`)
* Django CORS Headers
* Django Filter
* Token or JWT Authentication
* File Upload Support

The backend must be fully API-driven and integrate seamlessly with the React frontend.
