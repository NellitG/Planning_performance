### TypeScript Conversion

Convert the entire frontend codebase from JSX to TypeScript TSX.

Examples:

```text
Component.jsx  → Component.tsx
Page.jsx       → Page.tsx
Layout.jsx     → Layout.tsx
App.jsx        → App.tsx
index.jsx      → index.tsx
```

Requirements:

* Convert all React components to TSX.
* Add proper TypeScript interfaces and types.
* Type all props, hooks, state variables, API responses, and context providers.
* Remove JavaScript-only patterns where TypeScript alternatives are appropriate.
* Ensure all imports and exports are updated after conversion.
